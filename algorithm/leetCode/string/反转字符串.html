          # 344.反转字符串

          344open in new window

          编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 s 的形式给出。

          不要给另外的数组分配额外的空间，必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

          # reverse

          发现直接用reverse可以过

          var reverseString = function(s) {
   return s.reverse()
};

          # 双指针

          var reverseString = function(s) {
   for(let i=0,j=s.length-1;i<j;i++,j--){
       [s[i],s[j]]=[s[j],s[i]]
   }
};
