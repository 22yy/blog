              # HTTP3

              HTTP/3 把 HTTP 下层的 TCP 协议改成了 UDP, 不管顺序，也不管丢包的，不会出现 HTTP/1.1 的队头阻塞 和 HTTP/2 的⼀个丢包全部重传问题。

              1. 基于 UDP 的 QUIC 协议 可以实现类似 TCP 的可靠性传输。
                • 某个流发生丢包时，只会阻塞这个流，不影响其他
                • 头部压缩算法升级为QPack
